How to Install Fire.fm

Fire.fm is an add-on for the Mozilla Firefox web browser. It is compatible with all modern versions of Firefox, as well as SeaMonkey. You can get Firefox from Mozilla.

If you are already using Firefox, you can install Fire.fm by clicking here:

After installing Fire.fm and restarting Firefox, have a look in our quick guide on how to get started using Fire.fm. It's easy!
